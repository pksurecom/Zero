***

## 以下是一些个人分享:

#### 编辑器

- Sublime 

  名气响当当的编辑器,轻量,启动快.
  重点是插件多,个性化后可以满足大部分人的口味.
  推荐 Sublime Text3 的版本.

- vim

  编辑器之神.
  熟练使用后据说效率高的起飞,各种快捷键几乎无视鼠标的存在.
  但是学习曲线比较陡.
  
- Emacs

  神之编辑器.
  和 vim 齐名,常年被争论谁是最好的编辑器.
  PS: JS 是不是世界上最好的语言呢 ==
  
- HBuilder

  笔者刚入门的时候用的一个编辑器.
  适合新人使用,各种集成补全,在你刚入门没有把一堆架构工具用上去之前是很不错的选择.
